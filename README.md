# Git Cheat Sheet

Інтерактивна шпаргалка з Git командами (Basic / Branching / Remote) з пошуком та фільтрами.

## Як запустити
1. Відкрити `index.html` у браузері (подвійний клік або `xdg-open index.html`)
2. Використати пошук і фільтр категорій

## Features
- Basic commands
- Search
- Category filter

## Структура
- `index.html` — інтерфейс
- `styles.css` — стилі
- `app.js` — логіка пошуку/фільтрів/рендеру
- `data/commands.json` — база команд
