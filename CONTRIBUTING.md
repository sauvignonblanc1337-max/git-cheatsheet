# Contributing

Дякую за бажання зробити внесок!

## Як запропонувати зміни
1. Зробіть fork репозиторію
2. Створіть гілку: `feature/your-change`
3. Зробіть зміни та коміти у форматі Conventional Commits
4. Відкрийте Pull Request у `main`

## Правила
- Один коміт — одна логічна зміна
- Опис коміту має бути зрозумілим (feat/fix/docs/chore)
- Не додавайте секрети у репозиторій (файл `.env` ігнорується)
